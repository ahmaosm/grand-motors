# Проверка переводов

Сравни файлы `locales/ru.json` и `locales/en.json`:

1. Прочитай оба файла полностью
2. Сравни структуру ключей — все вложенные ключи должны совпадать
3. Найди:
   - Ключи которые есть в `ru.json`, но отсутствуют в `en.json`
   - Ключи которые есть в `en.json`, но отсутствуют в `ru.json`
   - Пустые значения ("") в любом из файлов
4. Выведи результат:
   - Если всё синхронно — сообщи что переводы в порядке
   - Если есть расхождения — покажи список с указанием пути к ключу (например `catalog.sort`)
5. НЕ вноси никаких изменений — только отчёт

# Предеплойная проверка

Выполни все проверки и выведи отчёт:

## 1. Сборка
- Запусти `npm run build` в папке `grand-motors/`
- Если есть ошибки — покажи их

## 2. Переводы
- Сравни ключи в `locales/ru.json` и `locales/en.json`
- Найди отсутствующие или пустые ключи

## 3. Изображения
- Прочитай массив машин из `constants/cars.ts`
- Для каждой машины проверь что все файлы из поля `images` существуют в `public/images/`
- Проверь что все изображения в формате WebP

## 4. Env-переменные
- Проверь наличие `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID` в окружении или `.env` файле

## 5. Отчёт
Выведи итог по каждому пункту:
- Сборка: OK / ОШИБКА
- Переводы: OK / список проблем
- Изображения: OK / список отсутствующих
- Env: OK / список отсутствующих

НЕ вноси никаких изменений — только отчёт.
